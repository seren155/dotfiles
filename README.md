# Dotfiles

Mi configuracion personas de archivos dots y los programas que acostumbro a utilizar en mi setup de **Arch Linux** + **Hyprland**

## Software Principal
* **WM:** Hyprland
* **Status Bar:** Waybar
* **Shell:** Fish
* **Terminal:** Kitty
* **Editor:** Nvim

## Reproduccion Multimedia
* **Video:** mpv
* **Musica:** Clementine
