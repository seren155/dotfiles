## 🛠️ Software & Core Components

| Category | Application | Description |
| :--- | :--- | :--- |
| **Compositor / WM** | [Hyprland](https://hyprland.org/) | Dynamic tiling Wayland compositor |
| **Terminal** | [Kitty](https://sw.kovidgoyal.net/kitty/) | Fast, GPU-accelerated terminal emulator |
| **Shell** | [Fish](https://fishshell.com/) | Smart, user-friendly interactive shell |
| **Text Editor** | [Neovim (Nvim)](https://neovim.io/) | Extensible, Vim-based text editor |
| **Status Bar** | [Waybar](https://github.com/Alexays/Waybar) | Highly customizable Wayland status bar |
| **App Launcher** | [Rofi](https://github.com/davatorium/rofi) | Window switcher and application launcher |
| **File Manager** | [Yazi](https://github.com/sxyazi/yazi) + Thunar | Blazing fast terminal file manager (with GUI fallback) |
| **Multimedia** | [MPV](https://mpv.io/) | Lightweight and powerful media player |
| **Wallpaper** | [Waypaper](https://github.com/anboman/waypaper) + swaybg | GUI frontend for managing desktop backgrounds |
| **Security** | [KeePassXC](https://keepassxc.org/) | Secure, offline password manager |
